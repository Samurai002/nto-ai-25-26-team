
Установка зависимостей (обязательно в папке с requirements.txt):

```
pip install -r requirements.txt
```

В папке data должны лежать следующие файлы:

- book_descriptions.csv
- book_genres.csv
- books.csv
- candidates.csv
- genres.csv
- targets.csv
- train.csv
- users.csv

Далее просто запустить все ячейки solution.ipynb
